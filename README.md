# Sms-Handy
JavaFx Project

in dem ein Handynetz simuliert wird, dessen Handys ausschließlich SMS senden können.

Das Projekt enthält zunächst die fünf fachlichen Modellklassen:

• SmsHandy
• PrepaidSmsHandy
• TariffPlanSmsHandy
• Provider
• Message.

Alle Klassen befinden sich in dem Package smsHandy.
